# finwise

A finance tracking application built in Python using PySide6 and sqlite3.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/codexagon/finwise.git
cd finwise
```

2. Create a virtual environment and activate it:

```bash
python -m venv virtualenv
source virtualenv/bin/activate # for Linux & MacOS
.\virtualenv\Scripts\activate # for Windows
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run:

```bash
python main.py
```
